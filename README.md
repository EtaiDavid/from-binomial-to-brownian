# From Binomial to Brownian Motion

Understanding option pricing through the convergence from the discrete binomial model to the continuous Black-Scholes model.

## Project Overview

This project demonstrates how the binomial model for option pricing converges to the Black-Scholes model as the number of periods increases. The goal is to develop a deep understanding of the mathematical connection between discrete and continuous-time models in derivatives pricing.
